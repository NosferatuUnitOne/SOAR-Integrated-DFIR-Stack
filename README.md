# SOAR-Integrated DFIR SOC Stack

This project extends my previous Wazuh Blue Team SOC Lab into a SOAR-integrated EDR-style detection and response pipeline.

The lab uses a single Linux endpoint monitored by Wazuh. Alerts generated by Wazuh are forwarded to Shuffle SOAR for parsing, threat intelligence enrichment, case creation, and response decision logic.

The project is designed to run on a single physical machine using multiple virtual machines, with a focus on lightweight, staged deployment.
